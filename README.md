# BaseDialog

#### 介绍
这是一个基于Android自带Dialog封装的一个Dialog调用框架，代码结构清晰，扩展性好。

#### 安装教程

项目的root Gradle下添加 maven { url 'https://jitpack.io' }，如下

```markdown
    repositories {
        google()
        jcenter()
        maven { url 'https://jitpack.io' }
    }

    allprojects {
        repositories {
            google()
            jcenter()
            maven { url 'https://jitpack.io' }
        }
    }
```
modle 的Gradle添加

```markdown
implementation 'com.github.daixuenan:BaseDialog:v1.0.1'
```
